# Controls
***This stuff needs to be confirmed before turnin***
## Player  
  - A = left
  - D = right
  - LeftShift = run
  - Space = jump
  - F = shoot
  - ? (question mark) = attack
  - , (comma) = block

## Game State
  - P = pause
  - L = reset

## Switching
  - M/N = item forward/backward
  - K/J = enemy forward/backward
